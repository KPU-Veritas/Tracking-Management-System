#1 인트로 화면 구성
- 중앙에 로고 배치
- 로고 하단에 간략한 어플리케이션 소개 텍스트

#2 로그인 & 회원가입 화면
- 로그인과 회원가입은 일반적인 서비스와 동일
- 회원가입 시 저번에 지적받은 개인정보 문제를 해결할 방안으로 특정 기관에서만 사용한다고 가정할 때 기관을 인증할 수 있는 코드를 입력
- 기관 인증 코드는 회원가입 시에 입력하거나 회원가입 때 입력하지 않고 서비스 최초 설치 시에 입력하는 방식도 괜찮을듯 함..

#3 메인화면
- 하단에 네비게이션 바로 메인화면/동선체크/환경설정 메뉴로 이동
- 최초 화면은 슬라이드 화면같이 '메인'이 출력되도록 함
- 상단에 현재 시간, 중앙에 블루투스 온오프 버튼, 버튼 하단에 서비스 안내 텍스트 배치
- 좌측이 블루투스가 켜져있는 상태 (서비스가 작동중), 우측이 블루투스가 꺼져있는 상태 (서비스 미작동중)
- 최상단에 (사용중인 기관명) 텍스트 배치

#4 동선 확인 메뉴 - 최근 동선 확인
- 최초로 들어가면 '최근 동선 확인', '최근 접촉 확인' 두 가지 중에서 하나를 선택할 수 있음
- 최근 동선 확인을 선택하면 최근 몇일간의 날짜를 리스트로 출력
- 특정 날짜를 선택하면 날짜 화면이 출력되고 날짜 화면 안에서는 비콘으로 데이터를 기록된 시간을 표기
- 시간을 선택하면 어떤 건물, 그 당시 그 장소의 밀접 접촉 여부, 확진자 발생 여부를 표시

#5 동선 확인 메뉴 - 최근 접촉 확인
- 최초 화면에서 우측 버튼 선택하여 접근
- 동선 확인과 동일하게 몇일간의 날짜를 리스트로 출력
- 날짜를 선택하면 그 날 내가 몇 차에 걸쳐서 접촉하였는지 출력됨

#6 환경설정
- 로그인 정보 : 회원 정보 확인, 로그아웃
- 블루투스 설정 : 블루투스 항상 켜기와 같은 블루투스 관련 설정
- 알림 설정 : 비콘 접촉 알림, 확진자와의 밀접 접촉 여부 알림 송수신 여부 설정
- 파일 관리 설정은 오타임 ㅎㅎ;
- 어플리케이션 정보 확인 : 어플리케이션 버전, 기관명 확인
- 고객센터 문의하기 : 서비스 고객센터 연락처 + 사용중인 기관의 관리자 연락처
