#1 로그인화면
- 관리자 계정 로그인 화면
- 관리자 계정은 따로 가입하는게 아니라 서비스 제공사에서 제공하는거니까 회원가입 대신 문의 버튼으로
- 문의버튼은 뭐 계정 분실 이럴때 쓰라고 만들어둔거고
- 밑에 접속 IP는 보통 이런거 관리 페이지에 떠서 넣었음

#2 메인화면
- 통합정보시스템이랑 유사하게 구성
- 상단 로고 옆에 메뉴를 누르면 좌측 사이드에 세부 메뉴에서 선택할 수 있도록 만듬

#3 회원정보
- 회원정보 관리 : 회원 가입시 작성한 데이터를 전부 출력하는 것으로, 확진 정보 기입 가능
- 확진 정보 기입 시 따로 팝업 경고 메시지 출력 -> #4번 슬라이드

#6 시설 정보
